Este sitio web fue creado con el objetivo de ser un punto de partida para planificar y desarrollar proyectos orientados con los Sistemas de Información Geográfica, esto por medio de la concentración de fuentes de datos oficiales. También se busca mostrar los alcances que tienen los datos para generar información y conocimiento.

Funciones:

La función del Geobuscador ("Inicio"), con fuentes de INEGI, CONABIO y ENCICLOVIDA, realiza una búsqueda directa o aproximada de datos geoespaciales con link directo para visualizar y descargar.

La página ("Datasets") concentra diversas fuentes de información vectorial y raster orientadas a Topografía, Incendios, Población, Flora, Fauna, Medio ambiente, Hidrología etc.

En caso de no encontrar la información en las fuentes de datos presentadas, buscar otras fuentes.
